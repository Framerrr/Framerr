# Themes

Framerr comes with several built-in themes and extensive customization options.

## Built-in Themes

| Theme | Description |
|-------|-------------|
| **Dark Pro** | Default dark theme with deep navy tones |
| **Noir** | Dark black and silver theme, great for OLED displays |
| **Nord** | Inspired by the popular Nord color palette |
| **Dracula** | Classic Dracula dark theme |
| **Catppuccin** | Warm, cozy Catppuccin Mocha palette |
| **Nebula** | Deep purple cosmic theme |
| **Light** | Clean, bright light theme |

## Changing Theme

1. Go to **Settings → Customization → Colors**
2. Select your preferred theme from the dropdown
3. Changes apply instantly
4. Themes are configured per user, across all devices

## Accent Colors

You can customize the accent color independently of the theme. The accent color is used for active elements, buttons, links, and highlights.

## UI Modes

Framerr offers different visual styles:

- **Glass** (default) — Modern glassmorphism with blur effects and translucent surfaces
- **Solid** — Clean, opaque surfaces without transparency — great for readability
- **Flatten** — Per widget configurable. Widgets backgrounds are removed to blend contentinto the wallpaper for a minimal, borderless look

# Widgets

Widgets are the building blocks of your Framerr dashboard. Each widget displays data from one or more integrations.

## Managing Widgets

### Edit Mode

Click **Edit Dashboard** (or the pencil icon) to enter edit mode where you can:

- **Add** widgets to the dashboard from the widget catalog
- **Drag** widgets to rearrange them
- **Resize** widgets by grabbing their edges
- **Configure** widgets by clicking the gear icon located on each widget
- **Remove** widgets from the dashboard

### Widget Configuration

Each widget has a settings panel where you can:

- Choose which integration instance to use
- Toggle visibility of specific data
- Set display preferences

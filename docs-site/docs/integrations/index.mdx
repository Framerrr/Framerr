## Adding an Integration

1. Navigate to **Settings â†’ Integrations â†’ Service Settings**
2. Click **Add Integration**
3. Select the service you want to connect
4. Fill in the required fields (URL, API key, etc.)
5. Click **Test Connection** to verify
6. Save the integration

## Multiple Instances

You can add multiple instances of the same integration type. For example, two Sonarr instances (one for TV, one for anime) â€” each gets its own connection and can be assigned to different widgets.

:::tip Instance Names
You can rename any integration instance from its settings page. Custom names appear in widget configuration dropdowns and notification messages â€” useful when running multiples of the same type (e.g., "Sonarr - Anime" vs "Sonarr - TV").
:::

## How Data Flows

```
Your Service (e.g., Plex) â†’ Framerr Server (proxy) â†’ Dashboard Widget
```

All API requests from widgets are proxied through the Framerr server. Your API keys and tokens are **never exposed to the browser** â€” they're stored encrypted on the server and used server-side only.

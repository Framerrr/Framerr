# Service Status

Displays the health and uptime status of your monitored services. Shows up/down status, response times, and uptime percentages in a card layout.

## Supported Integrations

| Integration | Features |
|-------------|----------|
| [Framerr Monitor](/docs/integrations/framerr-monitor) | Built-in HTTP/TCP monitoring — no external tools needed |
| [Uptime Kuma](/docs/integrations/uptime-kuma) | Full Uptime Kuma monitor data, heartbeat history |



## Configuration Options

This widget has no additional configuration options beyond the [common widget settings](/docs/widgets#common-widget-settings).

## Uptime History

Clicking any individual monitor card opens a popover showing its **24-hour uptime history** — a visual heartbeat timeline of up, down, and degraded states with response times.

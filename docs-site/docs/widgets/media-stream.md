# Media Stream

Displays currently active streaming sessions from your media server. See what's playing, who's watching, stream quality, and playback progress in real time.

## Supported Integrations

| Integration | Features |
|-------------|----------|
| [Plex](/docs/integrations/plex) | Active streams, transcode info, user, progress |
| [Jellyfin](/docs/integrations/jellyfin) | Active streams, user, progress |
| [Emby](/docs/integrations/emby) | Active streams, user, progress |



## Configuration Options

| Option | Default | Description |
|--------|---------|-------------|
| View Mode | Auto | **Auto** — adapts to widget size. **Carousel** — horizontal scroll through sessions. **Stacked** — vertical list of sessions. |
| Hide when no streams | On | Automatically hides the widget when nothing is playing |

# Tautulli Stats

Displays Plex watch history and library statistics from Tautulli. View top media per category and library analytics with a configurable stats bar.

## Supported Integrations

| Integration | Features |
|-------------|----------|
| [Tautulli](/docs/integrations/tautulli) | Watch history, top media, library stats |

## Configuration Options

| Option | Default | Description |
|--------|---------|-------------|
| Items to Show | 5 | Number of items displayed in lists. Options: **5**, **10**, **15**, **20**. |
| Stats Bar | Show | Show or hide the statistics summary bar |

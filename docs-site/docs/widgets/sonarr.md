# Sonarr Library

Displays your Sonarr TV show library with upcoming and missing episodes from your library.

## Supported Integrations

| Integration | Features |
|-------------|----------|
| [Sonarr](/docs/integrations/sonarr) | Series list, missing episodes, queue, stats |

## Configuration Options

| Option | Default | Description |
|--------|---------|-------------|
| View Mode | Auto | **Auto** — adapts layout to widget size. **Stacked** — sections stacked vertically. **Column** — sections in columns side by side. |
| Stats Bar | Show | Show or hide the statistics bar at the top (series count, episode count, etc.) |

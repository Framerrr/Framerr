# Downloads

A unified downloads widget showing active transfers from your download client. Displays download progress, speeds, ETA, and queue status.

## Supported Integrations

| Integration | Features |
|-------------|----------|
| [qBittorrent](/docs/integrations/qbittorrent) | Torrents, progress, speed, ETA |
| [SABnzbd](/docs/integrations/sabnzbd) | NZBs, progress, speed, ETA |



## Configuration Options

| Option | Default | Description |
|--------|---------|-------------|
| Stats Bar | Show | Show or hide the summary bar with total speed and queue count |

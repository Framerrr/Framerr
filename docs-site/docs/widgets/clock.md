# Clock

A simple, elegant clock widget. Displays the current time with optional date, seconds, and 24-hour format. No integration required â€” this is a standalone utility widget.

## Configuration Options

| Option | Default | Description |
|--------|---------|-------------|
| 24H | Off | Switch between 12-hour and 24-hour time format |
| Seconds | Off | Show seconds in the time display |
| Date | On | Show the current date below the time |
